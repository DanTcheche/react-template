# React + TypeScript + Vite

## Prerequisites

- [nvm](https://github.com/nvm-sh/nvm#installing-and-updating)


### Using nvm

1. Install the required Node.js version:
    ```sh
    nvm install
    ```

2. Use the required Node.js version:
    ```sh
    nvm use
    ```

## Installation

1. Install npm dependencies:
    ```sh
    npm install
    ```

## Running the Project

To run the project, use:
```sh
npm run dev
